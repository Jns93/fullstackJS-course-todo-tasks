# 🛠️ Abrir e rodar o projeto

# API (Dir: server)

1. Iniciar a api: **npx ts-node src/main.ts**

2. Executar os testes: **npx jest**

# Front (Dir: client)

1. Iniciar o servidor dev: **npm run dev**

2. Executar os testes: **npm run test**
